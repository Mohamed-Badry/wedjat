# UWE-4 Orbit Decay AI API

## Run locally

Install dependencies:

python -m pip install -r requirements.txt

Run the API:

python app.py

Open docs:

http://127.0.0.1:8000/docs

Main endpoints:

/health
/model-info
/latest-prediction
/predict/7
/predict/30
/dashboard

Important: Do not upload Space-Track credentials to GitHub.
